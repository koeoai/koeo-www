# Koeo Brand Assets

Official brand assets for Koeo. Use these assets consistently across all platforms and materials.

## Logos

| File | Description | Use Case |
|------|-------------|----------|
| `logo-gradient.svg/png` | Purple-to-magenta gradient logo | Light backgrounds |
| `logo-black.svg/png` | Solid black logo | Light backgrounds, print |
| `logo-white.svg/png` | Solid white logo | Dark backgrounds |
| `logo-on-gradient.svg/png` | White logo on gradient background | Standalone use, social media |

## Icons

Icon-only mark without wordmark. Available in multiple sizes for different use cases.

| File | Description |
|------|-------------|
| `icon-gradient.svg/png` | Gradient icon for light backgrounds |
| `icon-black.svg/png` | Black icon for light backgrounds |
| `icon-white.svg/png` | White icon for dark backgrounds |
| `icon-on-gradient.svg/png` | White icon on gradient background |

### Icon Sizes (PNG)

Each icon variant is available in these sizes:
- `*-favicon.png` - 16×16px (favicons)
- `*-small.png` - 32×32px (small UI elements)
- `*-medium.png` - 128×128px (app icons)
- `*-large.png` - 512×512px (high-res, social profiles)

## Banners

Neural network-themed banners with animated glow effects.

### Sizes
- `banner-wide-*` - 1500×500px (YouTube, LinkedIn cover, website headers)
- `banner-social-*` - 1200×630px (Twitter/X header, Open Graph)
- `banner-square-*` - 1200×1200px (Instagram, profile backgrounds)

### Variants
- `*-logo` - Centered Koeo logo
- `*-plain` - No logo, just neural network background
- `*-mantra` - Tagline "Any model, one place, no infra to chase." with logo

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Purple | `#7C3AED` | Main brand color, CTAs, links |
| Deep Purple | `#4C1D95` | Dark accents, footer backgrounds |
| Magenta | `#E02F87` | Accents, gradient endpoints |
| Light Pink | `#F472B6` | Hover states, highlights |

### Primary Gradient
```css
background: linear-gradient(135deg, #7C3AED, #E02F87);
```

## Typography

- **Logo Font**: Outfit (weight: 900, letter-spacing: -0.03em)
- **Body Font**: Inter

## Usage Guidelines

1. Maintain clear space around the logo equal to the height of the icon
2. Do not stretch, rotate, or alter the logo proportions
3. Use appropriate color variants for background contrast
4. Minimum logo width: 100px for digital, 25mm for print
